Основные принципы дизайна:

  • Боковая панель слева (240px) - действия + список файлов
  • Кнопка "Show Graph" выделена - градиент фиолетовый, самый заметный элемент
  • Контекстные кнопки - появляются только для .xsd файлов
  • Бейджи - оранжевая точка для dirty файлов, красный кружок с количеством ошибок
  • Плавные переходы - анимация при смене файла, hover эффекты на кнопках
  • Мягкие тени и скругления - 6-8px для кнопок, box-shadow для глубины
