# Altova XMLSpy - Функции для работы с XML и XSD схемами

## Что такое XMLSpy?

**Altova XMLSpy** — это ведущий в отрасли редактор XML и среда разработки, используемая более чем 5 миллионами разработчиков по всему миру. Это AI-готовый редактор JSON и XML для моделирования, редактирования, преобразования и отладки всех связанных технологий.

---

## Функции для работы с XML

### Режимы просмотра и редактирования

- **XML Text View** (Текстовый режим) — Текстовое редактирование XML с интеллектуальными помощниками ввода
- **XML Grid View** (Табличный режим) — Графическое иерархическое представление на основе контейнеров с табличным представлением повторяющихся элементов
- **XML Viewer** — Удобное визуальное представление XML
- **XML Generator** — Генерация экземпляров XML из схем

### Валидация и проверка

- **XML Validator** — Самый соответствующий стандартам валидатор XML в индустрии
- **Validation on RaptorXML Server** — Сверхбыстрая валидация, соответствующая стандартам W3C
- **XML Syntax Checking** — Проверка синтаксиса в реальном времени

### Инструменты преобразования и запросов

- **XSLT Editor** — Контекстно-зависимые помощники ввода, окно структуры XSL
- **XSLT Debugger** — Пошаговая отладка сложных таблиц стилей (поддержка XSLT 1.0, 2.0, 3.0)
- **XSLT Profiler** — Оптимизация производительности XSLT
- **XSL Speed Optimizer** — Революционный запатентованный подход для ускорения преобразований
- **XSLT Back-mapping** — Корреляция выходных элементов с исходными узлами
- **XPath Builder & Tester** — Построение и тестирование выражений XPath
- **XPath Debugger** — Пошаговая отладка вычислений XPath
- **XQuery Editor** — Полная поддержка XQuery 1.0 и 3.1
- **XQuery Debugger** — Отладка выражений XQuery
- **XQuery Profiler** — Анализ и оптимизация производительности XQuery
- **XQuery Back-mapping** — Сопоставление результатов XQuery с исходными документами
- **XQuery Update Facility Editor** — Составление и выполнение выражений XQUF

### Сравнение и анализ

- **Compare XML** — Сравнение и слияние XML файлов
- **XML Chart Generation** — Создание диаграмм из данных XML

## Функции для работы с XSD (XML Schema)

### Редактирование XSD

- **Graphical XSD Editor** — Визуальное редактирование схем методом drag-and-drop
- **XML Schema 1.0 и 1.1 Support** — Полная поддержка обеих версий
- **Intelligent Entry Helpers** — Контекстно-зависимые выпадающие меню
- **Advanced Identity Constraint Editing** — Работа с ограничениями unique, key, keyref
- **Smart Restrictions Support** — Визуальное редактирование ограничений
- **Aggregated XML Schemas** — Поддержка includes/imports схем

### Генерация XSD

- **Generate XSD from XML Instance** — Создание схемы из одного или нескольких XML файлов
- **Database to XSD Generation** — Генерация XSD из схем баз данных
- **DTD to XSD Conversion** — Преобразование DTD в XML Schema
- **Generate XSD from JSON Schema** — Преобразование JSON Schema в XSD

### Преобразование XSD в код/данные

- **XSD to XML Generation** — Генерация примеров экземпляров XML из XSD
- **XSD Documentation Generation** — Автоматическая генерация документации схемы
- **XSD to Java/C#/C++ Code Generation** — Генерация кода и связывание данных
- **Database Generation from XSD** — Создание структур баз данных из схем
- **JSON Schema Generation from XSD** — Преобразование XSD в JSON Schema

### Валидация XSD

- **XSD Validation** — Валидация XML файлов по схемам XSD
- **Schema-Aware Editing** — Интеллектуальное автодополнение кода на основе схемы

## Особенности XML Grid View

- Автоматическое отображение изображений
- Легкое создание XPath фильтров
- Формулы XQuery для вычисления результатов
- Вставка содержимого из внешних приложений
- Настраиваемая группировка узлов для очень больших XML файлов

## Особенности JSON Grid View

- Революционный графический редактор JSON
- Поддержка JSON, JSON5, JSON Lines, JSON с комментариями (JSONC)
- Создание диаграмм из данных JSON
- Преобразование JSON в XML и наоборот

## XBRL инструменты

- Графический редактор таксономий XBRL
- Поддержка XBRL 2.1, XBRL Dimensions и XBRL Formula
- Редактирование и выполнение XBRL Table Linkbase
- Контекстно-зависимые вкладки и помощники XBRL
- Мастер таксономий XBRL для расширения US-GAAP или IFRS
- Генерация документации таксономии XBRL
- Валидация таксономии и экземпляров XBRL
